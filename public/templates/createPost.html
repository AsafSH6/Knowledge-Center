Posting Guide

  • Add code:
    use the tag <code> and specify the programming language.
    For example:
    <code python>
    class Example(object):
        def __init__(self):
            print 'Example'
    </code>

  • Add link:
    use the tag <link=url> where url is the specific link address.
    For example:
    <link=www.google.com>
